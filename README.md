# 🧮 Calculator Web App

A modern, responsive calculator built using **HTML, CSS, and JavaScript**.  
It supports basic arithmetic operations along with a history feature and keyboard input.

---

## 🚀 Features

- ➕ Basic operations (Addition, Subtraction, Multiplication, Division)
- 🧾 Calculation History
- 🗑️ Clear History option
- ⌨️ Keyboard support (Enter, Backspace, Numbers, Operators)
- 🎨 Modern UI (Glass + Neon design)
- 📱 Responsive (works on mobile & desktop)

---

## 🛠️ Tech Stack

- HTML5  
- CSS3  
- JavaScript 

## ⚙️ How to Run

1. Download or clone this repository  
2. Open the folder  
3. Double-click `index.html`  

👉 The calculator will open in your browser

---

## ⌨️ Keyboard Shortcuts

|-----|--------|
| 0–9 | Numbers |
| + - * / | Operators |
| Enter | Calculate |
| Backspace | Delete last |
| C | Clear display |
